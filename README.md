# PDF Merger
A modern, client-side PDF and text document merging tool built with **React + TypeScript**.  

## Objective
Users can upload multiple files, review them in a structured list, reorder or manage them, and merge them into a single output document.

This project focuses on **clean UI/UX**, **semantic structure**, and **practicing frontend architecture**.

---
## Features
- Upload multiple files at once
- Drag-and-drop upload interface for better UX
- Visual file list with metadata about files
- Merge-ready document preview list to second your action
